# 📱 To-Do App (React Native)

## 🚀 Features

* Add / Delete Tasks
* Mark tasks as Done
* Task filtering (Pending, Done, Deleted)
* Local storage using AsyncStorage

* React Native
* JavaScript
* AsyncStorage
